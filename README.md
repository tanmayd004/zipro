# zipro
## Run Locally

INSTALLATION [ TERMUX APP --ANDROID ]

```bash

  git clone https://github.com/tanmayd004/zipro.git

```

```bash

  cd zipro

```

```bash

  chmod +x zipro.sh

```

```bash

  ./zipro.sh

```
